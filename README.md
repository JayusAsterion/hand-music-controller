# Hand Music Controller

Aplicación web interactiva para controlar música electrónica/tecno usando gestos de manos detectados por cámara.

## Stack

- Vite
- React
- TypeScript
- MediaPipe Tasks Vision
- Tone.js
- Three.js
- React Three Fiber
- Zustand
- Tailwind CSS

## Instalación

```bash
npm install
npm run dev
